Eligibility checker

A non-binding pre-screen, run against one chosen scheme — it compares the member's ERP membership + milk history to that scheme's rules and returns plain-language reasons. It is a guide, never an approval: only the DCS board + bank decide. Settled mock = the spec for GET /cattle-induction/eligibility?scheme=….

        Rules this mock encodes (→ backend spec):

        • Eligibility is scheme-scoped — the same member can be eligible for one scheme and not another (thresholds differ per scheme's rules_json).

        • advisory:true always — the response is never a sanction (rules propose, humans dispose).

        • Membership tenure from coop_memberships.joined_on; milk from the ERP — if the ERP is degraded, the reason says so and the check still opens.

        • Backend delta: return a structured checks[] (label · ok · detail · source) so the app renders ticks, not a sentence list.
